Integrates Gravity Forms with Proposal Software Bidsketch.

== Description ==

[Bidsketch](https://www.bidsketch.com) helps you create professional looking proposals in minutes. Using professionally written and designed proposal templates, you can customize proposals and send them out for instant approval. Instant approvals are easy with the built in electronic signature feature. Bidsketch also makes up-selling easy by letting you mark specific fees as optional and giving clients the option to accept them when viewing a proposal.

This plugin makes it dead simple to automatically add leads that come in with your Gravity Forms form to Bidsketch (as clients).

== Installation ==

1. Upload the bidsketch-gravity-forms folder to the /wp-content/plugins/ directory
1. Activate the Gravity Forms Bidsketch Add-On for Gravity Forms plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the Gravity Forms settings menu, and selecting Bidsketch from the submenu
